朋友，您好 ~


15 年前的今天，那时的我正好在高三，准备着人生中的第一场抉择——高考，殊不知，那是在最无知的时候需要做重要的人生决定。

清楚的记得我们的教室在一楼，午休之后，试图放松一下午休后懒散的神经，一般都需要唱歌，不记得当时是什么歌曲了。大家正在唱歌的时候，突然，地面开始摇晃，一开始还以为是错觉，但摇晃越来越大，很快就无法站稳。

清楚的记得，我们的教室就在一楼。午休过后，同学们总喜欢找些方式来放松，唱歌就是常用的办法之一，不过具体当时在唱什么歌现在已经记不太清了。 

就在大家试图用轻快的歌声驱走午后倦意的时候，地面突然开始左右的摇晃。起先还以为是自己眼花，但是摇晃的幅度越来越大，很快我们便无法稳稳站立。

“地震!”有人喊道，我们一拥而出跑到教室外面的空地上，地面像波浪一样起伏，教学楼顶楼也左右的晃动，加上教室旁边有一块树林，树叶莎莎的声音，场面十分骇人。

时间飞逝，15 年过去了……


## 创作

写作是一条漫长的路，要有持之以恒的精神和毅力。很多人会发现，写作灵感和素材常来源于生活中的琐事和小细节。

要做一名出色的作者，你需要学会从生活的每一个方面汲取营养，让所有的生活体验为你的创作服务。

可惜我不是一个出色的作者，写不出那种惊涛骇浪的感觉，还是自己的文笔差了点意思。 


和写作有点类似的就是创业，要意识到你的全部人生，都理应要为你的创作提供养分，为它服务。

如何找到适合自己的项目：
第一，主动把自己的评估下调一个档次。
第二，选项目时设定一个最重要标准，其他条件作为代价来满足这个标准。
第三，从 65 分的项目开始，先赚第一块钱，逐渐积累技能和经验，会看到更好的机会。
总之，要让自己配得上好项目或对象，先从现实出发，从可实现的开始，慢慢磨练和发展，才可能找到最适合自己的选择。


如果您喜欢这份 Newsletter，[请转发给您朋友，](https://chasays.zhubai.love/)，这是对我最大的帮助，继续进步，继续给大家带来有价值的分享。

- [早发现早治疗并非总是对的，健康人群的体检筛查需要确切的证据](https://mp.weixin.qq.com/s/sFXhVTsXGwbrgeYYSxp8Gg)

乳腺癌、胃癌、大肠癌、肝癌、肺癌和食管癌等筛查，效果也未有定论，而对前列腺癌和甲状腺癌的筛查则明显是弊大于利。
美国疾控中心推荐的筛查项目也仅有四种（是不是少得出乎你的意料？）：乳腺癌，宫颈癌，肺癌，结直肠癌。
当有人建议你做某些体检项目时，你不妨问问对方：`这些检查能让我获益么？风险和代价是什么？`
如果对方能把问题回答清楚，那你不妨权衡利弊之后做选择，如果对方只谈获益，完全不提弊端，那么该机构的技术水平职业道德，就非常令人生疑了。

# 工具、资源推荐



- [羽毛球比赛编排](https://raw.githubusercontent.com/chasays/mdPicGo/master/win2022/%E5%8D%95%E5%BE%AA%E7%8E%AF%E6%AF%94%E8%B5%9B%E8%BD%AE%E6%AC%A1%E5%9C%BA%E6%AC%A1%EF%BC%8D%E8%87%AA%E5%8A%A8%E7%BC%96%E6%8E%92.xlsx)

在我国的球类比赛中通常采用的单循环的编排方法，基本上是固定左上角，逆转一个位置产生下一轮次的比赛秩序。这种编排无论参赛队数多少，都能保证一二队在最后一轮遇。当参赛队数为奇数时，末位补“0”，以同样方法进行编排。这种编排方法的大缺点就是倒数第二个队总是被轮空队“踪追击”。这种编排方法大家都比较熟悉，故轮次表从略。

- [CLUEbenchmark](https://github.com/CLUEbenchmark/SuperCLUE)


中文通用大模型基准（SuperCLUE），是针对中文可用的通用大模型的一个测评基准。

它主要回答的问题是：在当前通用大模型大力发展的背景下，中文大模型的效果情况，包括但不限于"这些模型不同任务的效果情况"、"相较于国际上的代表性模型做到了什么程度"、 "这些模型与人类的效果对比如何"。

它尝试在一系列国内外代表性的模型上使用多个维度能力进行测试。SuperCLUE 是中文语言理解测评基准（CLUE）在通用人工智能时代的进一步发展。


- [chatgpt-on-wechat](https://github.com/zhayujie/chatgpt-on-wechat)

使用 ChatGPT 搭建微信聊天机器人，基于 GPT3.5/4.0 API 实现，支持个人微信、公众号、企业微信部署，能处理文本、语音和图片，访问操作系统和互联网。
