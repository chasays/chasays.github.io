# 数字计算

## divmod

## abs
提到数字计算还有用的最多的， `abs`，比如求任意一个数的x进制，如果是复数就可以用上这个 `abs，` 对应的加上一个 `signal -`。

```py
import string
digs = string.digits + string.ascii_letters


def numberToBase(x, base):
    if x < 0:
        sign = -1
    elif x == 0:
        return digs[0]
    else:
        sign = 1

    x *= sign
    digits = []

    while x:
        digits.append(digs[int(x % base)])
        x = int(x / base)

    if sign < 0:
        digits.append('-')

    digits.reverse()

    return ''.join(digits)
```
但是还可以用切片的方式来处理这个算法，
```py
def numberToBase(n, b):
    if n == 0:
        return [0]
    digits = []
    while n:
        digits.append(int(n % b))
        n //= b
    return digits[::-1]
```
